## 设计一个栈数据结构

栈：一种后进先出的数据结构

栈的应用包括：
- 函数调用时，主函数传给子函数的参数先进栈，进入子函数后，子函数的局部变量也按序在栈中建立
- 子函数返回时，局部变量出栈、参数出栈。

栈类的成员：
stack类封装了栈的存储空间并提供了操作栈的函数。
具体设计如下：

---

**StackOfIntegers**

---
- elements: int[100]     //存放栈元素的数组

- size: int                       //栈内元素数量

---

+ StackOfIntegers()        //空栈的构造函数

+ empty(): bool              //判栈空否

+ peek(): int                   //读取栈顶元素而不移除

+ push(value: int): int     //将一个数据压栈

+ pop(): int                    //弹出一个数据

+ getSize(): int               //回栈内元素数量
